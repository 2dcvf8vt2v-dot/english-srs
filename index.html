First setup

English SRS works best in this order:

  1. Generate words JSON with GPT.
  2. Import the words JSON.
  3. Generate exercise pack JSON.
  4. Import the exercise pack JSON.
  5. Learn with Review and Practice.
  6. Export a backup sometimes.

Add words

Paste a word list into Generate with GPT, add a source name, then copy the words prompt.

Send the prompt to GPT. GPT should return a words JSON.

Paste that JSON into Import data and press Import JSON.

Add exercises

After importing words, check Exercise coverage.

If some words still need exercises, choose a batch size and copy the missing exercises prompt.

Send it to GPT, then import the returned exercise pack JSON.

Learn every day

Review uses spaced repetition cards.

Smart Practice chooses useful exercises from selected packs.

Weak Practice focuses on words you recently missed.

Pack Practice starts one specific exercise pack.

Fix weak words

Weak words are based on recent Practice answers, not your whole history.

A word can become weak if you miss it recently or your recent accuracy is low.

If your last 3 answers for a word are correct, it leaves Weak words.

Use Copy weak words prompt to generate fresh exercises for weak words.

Backup and restore

Your data is stored locally on this device. It does not sync automatically.

Export backup saves your words, exercise packs, exercises, attempts, and progress.

To restore data, import the backup JSON through Import data.

Offline and updates

The app works offline after it has loaded once from the hosted site.

When the app updates, open it with internet, wait a moment, close it, and open it again.

Your words and progress should stay on the device.
