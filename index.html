Basic flow

The app works in two layers: word cards for Review and exercise packs for Practice.

  1. Add words with Words generator or manual entry.
  2. Import the words JSON.
  3. Create exercises with Exercises generator.
  4. Import the exercise pack JSON.
  5. Use Review to remember meanings over time.
  6. Use Practice to test words in context.
  7. Run audits when definitions or exercises start looking messy.
  8. Export a backup before big changes.

Review is based on due dates. Practice uses imported exercises plus dictionary questions generated from your word cards.

Words generator

Words are the base of everything. Each card needs a word, short meaning, example, tags, and source.

Paste words, phrases, or book context into Words generator, set a source name, then copy the prompt.

Send the prompt to GPT and import the returned words JSON through Import data.

Good cards use short English-only meanings and natural examples. Examples are for the card, not for future exercises.

New words per day controls how new imports are paced. If you already imported a large dictionary, use Reschedule new words in Settings.

Exercises generator

Exercises generator creates GPT prompts for practice packs. It uses the words you pasted in Words generator or words selected by Missing exercises / Weak words.

You can choose which GPT exercise types to generate: Sentence choice, Fill blank, Paraphrase, Best context, and Tone.

Dictionary questions are different. The app creates choose_meaning and choose_word questions automatically from your word cards, so GPT should not generate them.

Use Exercise pack when you want a normal new practice pack. Use Missing exercises when some words have no imported exercises yet. Use Weak words when you want fresh practice for words you recently missed.

Practice modes

Review is spaced repetition. It shows words when they are due and updates their next due date after your rating.

Smart Practice picks useful exercises from selected packs and mixes in generated dictionary questions. It is automatic and word-priority based.

Weak Practice focuses on words you recently missed. A word leaves Weak words after recent correct answers.

Focused Practice is the manual mode. Choose exercise types, tags, source, and session size yourself.

Pack Practice starts one imported pack directly and uses the exercises from that pack.

Focused Practice filters

Type lets you choose one or more exercise groups. For example, you can train only Tone, only Best context, or several types together.

Tags shows common tags only, so the list does not turn into a huge mess. Selected tags use OR logic: a word can match any selected tag.

Source narrows practice to one book, batch, article, or other source.

Matches previews how many imported exercises and dictionary words fit your current filters before you press Go.

Weak words

Weak words are based on recent Practice answers, not your whole history.

A word becomes weak when you miss it recently, miss it several times, or your recent accuracy for that word is low.

If the last 3 answers for a word are correct, it leaves Weak words.

Use Weak Practice for quick repair sessions. Use the Weak words prompt when you want GPT to generate fresh context exercises for those words.

Exercise audit

Exercise audit finds exercises that reuse card examples or look too similar to them.

Exact matches are usually safe to delete in bulk because they copy the card example too directly.

Similar matches need manual review. Select only the suspicious ones, then delete selected.

Exercise audit deletes exercises only. It does not delete words, sources, packs, or Review progress.

Word audit and fixes

Word audit checks card quality: missing examples, missing tags, long meanings, Russian text, duplicate meanings, and meanings that reuse the target word too closely.

Select the cards or duplicate groups you want to fix, then copy the GPT fix prompt.

GPT should return a wordFixes JSON. Import it with Paste & import fixes on the Word audit screen, or through Import data.

Word fixes keep the same card IDs. They can update meaning, example, tags, and source without resetting Review progress.

Import data

Import data accepts words, exercise packs, word fixes, and full backups. The app detects the JSON type automatically.

You can paste JSON manually, choose a JSON file, or use Paste & import from clipboard when GPT's response is already copied.

If clipboard import fails, paste the JSON into the field and press Import JSON. Browser clipboard access can be picky, especially outside secure contexts.

If import fails, read the error. Most problems come from extra text outside JSON, broken quotes, missing commas, or words that do not exist yet.

Backup and restore

Your data is stored locally on this device. It does not sync automatically.

Export backup saves words, exercise packs, exercises, practice attempts, review attempts, and activity history.

To restore data, import the backup JSON through Import data.

Make a backup before Clear all data, bulk audit deletion, large imports, or risky testing.

Offline and updates

The app works offline after it has loaded once from the hosted site.

Use Refresh app after updates or when the installed app still shows an old cache version.

Your words and progress should stay on this device after refresh. Still, exporting a backup before bigger changes is the sane move.
